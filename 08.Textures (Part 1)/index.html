Mipmaps generated automatically.

The above result can be achieved with two settings:

  • gl.TEXTURE_MIN_FILTER set to gl.LINEAR
  • gl.TEXTURE_MAG_FILTER set to gl.LINEAR

Pixels aren't blended thanks to gl.TEXTURE_MAG_FILTER set to gl.NEAREST

Pixels are flipped vertically thanks to gl.UNPACK_FLIP_Y_WEBGL set to true
